# Discord Token Checker v1.0

Gelişmiş Discord Token kontrol ve analiz aracı. Çoklu thread desteği, proxy desteği ve detaylı raporlama özellikleriyle.

## 🚀 Özellikler

- ✅ Çoklu thread desteği (50-100+ thread)
- ✅ Proxy desteği
- ✅ Nitro kontrolü
- ✅ Detaylı token analizi
- ✅ GUI arayüzü
- ✅ JSON/TXT çıktı formatları
- ✅ E-posta ve telefon bilgisi
- ✅ Hesap oluşturulma tarihi
- ✅ 2FA kontrolü
- ✅ İlerleme çubuğu
- ✅ Detaylı istatistikler
